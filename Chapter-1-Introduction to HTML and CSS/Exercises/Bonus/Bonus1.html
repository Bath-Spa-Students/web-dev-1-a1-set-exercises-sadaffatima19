Paragraph Element

<p> …paragraph goes here… </p>

- Groups sentences and sections of text together.

Line Break Element

…text goes here <br>
This starts on a new line….

- Causes the next element or text to display on a new line

Blockquote Element

<blockquote>
…text goes here… <blockquote>

- Indents a block of text for special emphasis

Bold Text

<b>...text goes here...</b>

- Text has no extra importance but is styled in bold font by usage and convention.

Emphasized Text

<em>...text goes here...</em>

- Causes text to be emphasized in relation to other text; usually displayed in italics.

Italicized Text

<i>...text goes here...</i>

- Text has no extra importance but is styled in italics by usage and convention.

Mark Text

<mark>...text goes here...</mark>

- Text that is highlighted in order to be easily referenced.

Small Text

<small>...text goes here...</small>

- Legal disclaimers and notices displayed in small font-size.

Strong Text

<strong>...text goes here...</strong>

- Strong importance; causes text to stand out from surrounding text; usually displayed in bold.

CSS Resource Guide

Configuring Text with CSS

Font-Weight

/* Single values */
font-weight: normal;
font-weight: bold;
font-weight: 400;

- Configures the boldness of text

Font-Style

h3{font-style: normal;}

- Configures text to an italic style

Font-Size

h3{font-size: 16px;}

- Configures the size of the text

Font-Family

h3{font-family: "Times New Roman", Times, serif;}

- Configures the font typeface of the text

Text-Align

h3{text-align: center;}

- Configures alignment of text within a block display element

Text-Indent

h3{text-indent: 50px;}

- Configures the indentation of the first line of text

Text-Decoration

h3{text-decoration: overline;}

- Modifies the appearance of text with an underline, overline, or line-through

Text-Transform

h3{text-transform: uppercase;}

- Configures the capitalization of text

Letter-Spacing

h3{letter-spacing: 2px;}

- Configures space between text characters

Word-Spacing

h3{word-spacing: 30px;}

- Configures space between words

Text-Shadow

h3{text-shadow: 2px 2px #ff0000;}

- Configures a drop shadow on text

Background Color

body {background-color: blue;}

- Gives a backgorund color to the text
